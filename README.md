## Contributing

### Local Setup

Clone the repository and run on the root folder:

```
pnpm i
pnpm run dev
```

#### Scripts available:

Run the scrips with `pnpm run xxx`.

1. `dev`: Start dev server
2. `build`: Build for production
3. `lint`: Lint code
4. `lint:fix`: Lint code with auto fixing
